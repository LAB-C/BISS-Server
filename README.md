# transaction address #
## iot 기기 등록 ##
url : hack.cielsoft.me/api/iot/regist (PUSH)
### request ###

```json
{
    "uuid": "iot 고유번호",
    "key": "엔드 디바이스 지갑 값",
    "name": "iot 기기 이름",
    "desc": "iot 기기 설명"
}
```
## iot 등록 확인 ##
url : hack.cielsoft.me/api/iot/regist/{uuid} (GET)

### response ###
```json

    "success" : true,
```
